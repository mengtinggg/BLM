Link to Source Code with raw data files: https://drive.google.com/file/d/1kM597SZCsJ3idiegV50TRHQtt1slEC8E/view?usp=sharing
Data was removed from Instagram and Twitter to reduce file size. 

File Contains 4 Folders:

- Facebook
- Instagram
- Reddit
- Twitter

--------------------------------------
Facebook Folder Contains:
--------------------------------------
1. Facebook_Data.json - Data collected from Facebook
2. Facebook_TextVisualize.ipynb - Jupyter Notebook to clean and transform data into WF WordCloud and IDF WordCloud
3. Facebook_TopicModeling.ipynb - Jupyter Notebook to model topics using LDA model
4. FacebookIDFCloud.png - Image of IDF WordCloud
5. FacebookLDAModel.html - Topic Model Visualization using pyLDAvis
6. FacebookScraper.py - Source code to extract data from Facebook
7. FacebookWFCloud.png - Image of WF WordCloud

--------------------------------------
Instagram Folder Contains:
--------------------------------------
1. BlackLivesMatter_InstagramComments.json - Combined file of all Instagram Comments collected
2. BlackLivesMatter_InstagramPosts.json - Combined file of all Instagram Posts Collected
3. Instagram_TextVisualize.ipynb - Jupyter Notebook to clean and transform data into WF WordCloud and IDF WordCloud
4. Instagram_TopicModeling.ipynb - Jupyter Notebook to model topics using LDA model
5. InstagramIDFCloud.png - Image of IDF WordCloud
6. InstagramLDAModel.html - Topic Model Visualization using pyLDAvis
7. InstagramPostCombiner.py - Source code to combine individual posts and comments into a single json file
8. InstagramScraper.py - Source code to extract data from Instagram
9. FacebookWFCloud.png - Image of WF WordCloud
10. Original_Data_blacklivesmatter - Zipped file of individual instagram post and comments

-------------------------------------
Reddit Folder Contains:
-------------------------------------
1. 2015_to_2016.png - Line Graph of Post Count against Time from 2015 to 2016
2. 2016_to_2017.png - Line Graph of Post Count against Time from 2016 to 2017
3. 2020_onwards.png - Line Graph of Post Count against Time from 2020 onwards
4. May2020.png - Line Graph of Post Count against Time during May2020
5. Reddit_TextVisualize.ipynb - Jupyter Notebook to clean and transform data into WF WordCloud and IDF WordCloud
6. Reddit_TopicModeling.ipynb - Jupyter Notebook to model topics using LDA model
7. Reddit2015andBefore.csv - Reddit data from 2014 to 2015
8. Reddit2018to2015.csv - Reddit data from 2015 to 2018
9. Reddit2019.csv - Reddit data from 2019
10. RedditIDFCloud.png - Image of IDF WordCloud
11. RedditJan2020_July2020 - Reddit data from Jan 2020 to July 2020
12. RedditLDAModel.html - Topic Model Visualization using pyLDAvis
13. RedditPostCount.ipynb - Jupyter Notebook to plot line graph of post count against time
14. RedditScriptScraper.py - Source code to extract data from Reddit
15. RedditWFCloud.png - Image of WF WordCloud
16. TimelineOverview.png - Line Graph of Post COunt against Time from all data collected

Twitter Folder Contains:
-------------------------------------
1. defundThePolice.txt - Data collected from Twitter with the query #defundthepolice
2. portlandprotest.txt - Data collected from Twitter with the query #portlandprotest

4. Twitter_TextVisualize.ipynb - Jupyter Notebook to clean and transform data into WF WordCloud and IDF WordCloud
5. Twitter_TopicModeling.ipynb - Jupyter Notebook to model topics using LDA model

8. TwitterHashtags.gephi - Gephi file generated using Twitter Hashtags
9. TwitterHashtags.gml - .gml file to be used in Gephi
10. TwitterIDFCloud.png - Image of IDF WordCloud
11. TwitterLDAModel.html - Topic Model Visualization using pyLDAvis
12. twitterLimitScraper.py - Source code to extract data from Twitter
